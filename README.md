# Библиотекаи GO барои генератсия кардани формахои GF

Библиотекаи зеринро ба проекти худ ҳамроҳ карда, Шумо метавонед аз модел(структура)ҳои худ формаҳои **CRUD** ро генерақия кунед. 
Албатта библиотека танҳо барои сохтани инструктсияҳои форма мебошад худи формаҳо аз тарафи генератори **GF** сохта мешавад.
