# Библиотекаи GO барои генератсия кардани формахои GF

Библиотекаи зеринро ба проекти худ ҳамроҳ карда, Шумо метавонед аз модел(структура)ҳои худ формаҳои **CRUD** ро генератсия кунед. 
Албатта библиотека танҳо барои сохтани инструктсияҳои форма мебошад худи формаҳо аз тарафи генератори **GF** сохта мешавад.

## Пеш аз омӯхтани кор бо библиотекаи зерин, бо синтаксиси **GF** шинос шудан лозим аст.

Синтаксиси **GF** дар давоми корҳои барномасозон Ӯразов Б. ва Аъзамов Н. коркард карда шудааст. 
Максади асосӣ барои лоиҳаҳои оянда осон намудани, сохтан ва тағир додани формаҳои дохилкунӣ ва тағирдиҳии маълумотҳо, мебошад. 
Синтаксис дар асоси структураи **JSON** тартиб дода шуда, объекти асосии тартибдодашаванда **Page** мебошад. 

**Page** вохиди умумии элементи визуалӣ буда, контенти пурраи саҳифаро ташкил медиҳад ва дар дохили худ дигар элементҳои визуалиро нигоҳ медорад.
Сохтори асосии **Page** чунин мебошад:
```
{
  "form": {},
  "dataTable":{}
}
```
Чи хеле ки оварда шудааст. **Page** аз ду элементи асосӣ иборат мебошад. **Form**  ва **Data Table** .

**Form** барои нишон додани элементҳои кабулкунии маълумот тартиб дода мешавад. Хангоми холи будани структураи **Form** фақат қисми **Data Table** нишон дода мешавад.
Структураи **Form** чунин мебошад:
```
{
  "columns": [], //Массиви сутунҳо, дар дохили Form элементҳо бо воситаи сутунҳо ба тартиб дароварда мешавад.
  "submit": {} //Тугмаи фиристодан, Form танҳо як тугмаи фиристодан дорад. Ва ин тугма вобаста ба хосиятҳои худ функсияҳои гуногунро иҷро мекунад.
}
```
Дар навбати худ объекти **Column** аз маҷмӯи **Input** ҳо иборат мебошад. 

```
{
  "inputs":[] //Массиви элементҳои қабулкунӣ
}
```
Массиви элементҳои идоракунӣ дар худ ҳамаи элементҳои қабулкуниро дарбар мегирад. Структураи элементҳои қабулкунӣ барои ҳамаи элементҳо умумӣ мебошад ва чунин мебошад:

```
{
  "type": "",          // типи элементи идоракунӣ combo-box,date-time,text-view,number-view,check-box,label,search-view,text-field, hidden, auto-complete,file-uploader
  "name": "",          // идентификатори элемент, бо воситаи калиди зерин сохтори json тартиб дода мешавад
  "fromName": "",      // калиди махсус элемент бо кадом калид маълумотро кабул мекунад
  "readOnly": false,   // элемент танхо барои хондан аст
  "text": "",         // матне ки дар интерфейс пайдо мешавад
  "maxLength": 0,     // дарозии максималии элемент барои text-view,text-field, кимати максимали барои number-view
  "minLength": 0,     // дарозии минималии элемент барои text-view,text-field, кимати минимали барои number-view
  "isDefault": false, // пур кардани кимат хатми аст
  "metaData": "",     // барои search-view истифода мешавад, калиди маълумот мета
  "metaKey": "",      // барои search-view истифода мешавад,қимати маълумот мета
  "validMessage": "", // хабари система хангоми пур накардани маълумоти хатми
  "format": "",       // формати сана дар элементи date-time 
  "items": [],         // аъзоҳои элементи combo-box
  "searchObject": "",  // хангоми дар форма элементи search-view будан, аз кадом search-view гирифтани маълумотро нишон медихад
  "defaultValue": "",  // қимати пешфарзи(default) элемент
  "searchSource": "",  // суроғаи ҷустуҷӯ дар элементи search-view
  "dataType": "",       // типи кимати элемент асосан дар элементи search-view нишон дода мешавад
  "visible": "",       // калиди элементи checkbox ки хангоми актив будан элемент пайдо мешавад вагарна на
  "template": "",       // шаблони махсуси гирифтани кимат
  "fileSource": "",     // адреси махсусе, ки файлро сабт карда мешавад барои file-upoader 
  "fileExtensions": null, // расширениеҳои файлҳо, ки барои file-uploader иҷозат дода мешавад
  "fileMaxSize": 0      // андозаи максималии файле, ки барои file-uploader иҷозат дода мешавад бо байтҳо нишон дода мешавад
}
```
Тугмаи **Submit** вобаста ба ҳолати **Page** 2 вазифаро иҷро мекунад. Агар **Page** аз формаю ҷадвал иборат бошад, он гоҳ тугма вазифаи ҷустуҷӯро иҷро мекунад. 
Ҳангоми **Page** аз худи форма иборат будан, вазифаи Ивазкунии киматро иҷро мекунад.

Сохтори тугмаи форма **Submit**

```
{
      "text": "",              // матни тугма 
      "source": "",            // адресе ки малумотхо фиристонида мешавад
      "method": "",            // методи фиристодани маълумотҳо POST, PUT
      "confirmMessage": "",    // Хабаре, ки пеш аз фиристодани хабар пайдо мешавад. Агар холи бошад хабар намебарояд
      "lastAction": "",        // Амалҳое, ки бади иҷро шудани оператсия иҷро мешавад. success-message - хабари иҷро шудан,close - махкам кардани Page
      "successMessage": "",    // Хабар ҳангоми бо муваффаққият иҷро шудани дархост
      "type": ""               // Типи тугма. Ҳангоми бо ҷадвал будани Page метавонад функқияи ҷустуҷӯро иҷро кунад finder,loader
    }
```

**Data Table** барои нишон додани контенти маълумот истифода бурда мешавад. Хангоми холи будани структураи **Data Table** фақат қисми **Form** нишон дода мешавад.

Структураи объекти **Data Table** чунин мебошад. 

```
{
    "title": "",
    "keyColumn": "",
    "header": [],
    "pageSize": 0,
    "itemsCount": "",
    "delete": {},
    "edit": {},
    "add": {},
    "context": [],
    "indexes": [],
    "default_url": "/account/transaction/list",
    "exports": {},
    "top": "",
    "bottom": ""
  }
```
