# Библиотекаи GO барои генератсия кардани формахои GF

Библиотекаи зеринро ба проекти худ ҳамроҳ карда, Шумо метавонед аз модел(структура)ҳои худ формаҳои **CRUD** ро генератсия кунед. 
Албатта библиотека танҳо барои сохтани инструктсияҳои форма мебошад худи формаҳо аз тарафи генератори **GF** сохта мешавад.

## Пеш аз омӯхтани кор бо библиотекаи зерин, бо синтаксиси **GF** шинос шудан лозим аст.

Синтаксиси **GF** дар давоми корҳои барномасозон Ӯразов Б. ва Аъзамов Н. коркард карда шудааст. 
Максади асосӣ барои лоиҳаҳои оянда осон намудани, сохтан ва тағир додани формаҳои дохилкунӣ ва тағирдиҳии маълумотҳо, мебошад. 
Синтаксис дар асоси структураи **JSON** тартиб дода шуда, объекти асосии тартибдодашаванда **Page** мебошад. 

**Page** вохиди умумии элементи визуалӣ буда, контенти пурраи саҳифаро ташкил медиҳад ва дар дохили худ дигар элементҳои визуалиро нигоҳ медорад.
Сохтори асосии **Page** чунин мебошад:
```
{
  "form": {},
  "dataTable":{}
}
```
Чи хеле ки оварда шудааст. **Page** аз ду элементи асосӣ иборат мебошад. **Form**  ва **Data Table** .

**Form** барои нишон додани элементҳои кабулкунии маълумот тартиб дода мешавад. Хангоми холи будани структураи **Form** фақат қисми **Data Table** нишон дода мешавад.
Структураи **Form** чунин мебошад:
```
{
  "columns": [], //Массиви сутунҳо, дар дохили Form элементҳо бо воситаи сутунҳо ба тартиб дароварда мешавад.
  "submit": {} //Тугмаи фиристодан, Form танҳо як тугмаи фиристодан дорад. Ва ин тугма вобаста ба хосиятҳои худ функсияҳои гуногунро иҷро мекунад.
}
```

**Data Table** барои нишон додани контенти маълумот истифода бурда мешавад. Хангоми холи будани структураи **Data Table** фақат қисми **Form** нишон дода мешавад.
